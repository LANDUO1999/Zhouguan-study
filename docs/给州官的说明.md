## 🚀 在你电脑上开始使用的步骤

### 第一步：安装必要软件
1. 下载安装 Git：https://git-scm.com/downloads
2. 下载安装 VS Code（推荐）：https://code.visualstudio.com/
3. 安装后打开终端（Windows用CMD或PowerShell，Mac用终端），输入 `git --version` 确认安装成功

### 第二步：下载项目
有两种方式：

**方式A：从GitHub下载（推荐）**
```bash
git clone https://github.com/landuo1999/Zhouguan-study.git
cd Zhouguan-study
git checkout main
```

**方式B：直接复制现在的整个文件夹**
- 把服务器上的整个 `/root/.openclaw/workspace/` 文件夹复制到你电脑上

### 第三步：试试看！
1. 用浏览器直接打开 `index.html`，能看到网站就成功了
2. 打开 `docs/快速参考.md`，了解日常怎么操作
3. 想深入了解就看 `docs/本地开发指南.md`

## 📋 日常修改文章的流程（简化版）

1. 打开 `articles-database.json`，添加新文章信息
2. 打开 `index.html`，更新文章列表
3. 复制 `index.html` 到 `website/` 文件夹备份
4. 提交并推送到GitHub：
   ```bash
   git add index.html articles-database.json
   git commit -m "添加了新文章"
   git push origin main
   ```
5. 等5-10分钟，网站就自动更新了！
