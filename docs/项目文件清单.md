## 🎯 第一次使用时的检查清单

在你自己的电脑上开始使用前，请确认：

- [ ] 已安装 Git
- [ ] 已安装文本编辑器（VS Code推荐）
- [ ] 已克隆项目到本地
- [ ] 已切换到 `main` 分支
- [ ] 已阅读 `本地开发指南.md`
- [ ] 已保存 `快速参考.md` 到方便查看的地方
- [ ] 可以在本地浏览器中打开 `index.html` 预览网站

## 💾 备份建议

1. **每次修改前**：备份 `index.html` 和 `articles-database.json`

3. **云端备份**：GitHub已经是云端备份，记得定期 push
4. **本地备份**：可以复制一份到U盘或其他位置
