# 州官点灯网站 - 快速参考卡

## 🚀 5分钟快速开始

### 第一步：克隆项目
```bash
git clone https://github.com/landuo1999/Zhouguan-study.git
cd Zhouguan-study
git checkout main  # ⚠️ 必须在main分支！
```

### 第二步：修改文件
- 文章数据 → `articles-database.json`
- 网站页面 → `index.html`

### 第三步：提交并部署
```bash
git add index.html articles-database.json
git commit -m "描述你的修改"
git push origin main
```

---

## 📋 常用操作速查表

### 添加新文章流程
1. 在 `articles-database.json` 添加文章信息
2. 在 `index.html` 更新文章列表
3. 复制 `index.html` 到 `website/` 备份
4. 提交并推送到 GitHub

### 文章数据格式
```json
{
  "id": "20240401",
  "title": "文章标题",
  "url": "微信链接",
  "category": "cad",
  "date": "2024-04-01"
}
```

### 分类标签
- `cad` - CAD教程
- `su` - SU软件
- `adobe` - Adobe软件
- `ppt` - PPT技巧
- `ai` - AI工具
- `system` - 系统软件
- `gzh` - 公众号运营
- `life` - 个人/娱乐
