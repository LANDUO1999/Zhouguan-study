📖 完整使用指南

一

准备工作

确保你已经：
1. 安装了 Git
2. 从 GitHub 克隆了项目
3. 在 main 分支上

二

打开管理工具

双击项目文件夹里的 manager.html，用浏览器打开

三

添加文章

在"添加文章"标签页填写文章信息，然后下载更新后的数据库

四

生成网站

在"生成网站"标签页，选择数据库文件，生成新的 index.html

五

替换文件

用下载的文件替换项目里的旧文件

六

提交并推送

打开终端，运行：
git add index.html articles-database.json
git commit -m "更新文章"
git push origin main

Git 命令速查

# 查看状态 git status # 添加修改的文件 git add index.html articles-database.json # 提交更改 git commit -m "描述你的修改" # 推送到 GitHub git push origin main # 拉取最新更新 git pull origin main
