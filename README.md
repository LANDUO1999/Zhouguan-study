## 🚀 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/landuo1999/Zhouguan-study.git
cd Zhouguan-study
git checkout main
```

### 2. 本地预览
直接用浏览器打开 `index.html` 即可预览。

### 3. 部署到 GitHub Pages
```bash
git add index.html articles-database.json
git commit -m "更新网站"
git push origin main
```
