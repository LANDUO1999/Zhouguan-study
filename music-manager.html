📖 完整使用指南

一

添加新音乐

1. 在"添加音乐"标签页选择音乐文件
2. 填写歌曲名、艺术家、封面图标
3. 点击"添加到播放列表"

二

管理播放列表

在"播放列表"标签页可以：
- 查看所有音乐
- 删除不需要的音乐
- 调整播放顺序（拖拽）

三

下载文件

在"生成更新"标签页：
1. 下载 playlist.json
2. 复制音乐文件到 music/ 文件夹

四

更新 index.html

重要！还需要手动更新 index.html 里的播放列表：
1. 点击"生成 index.html 代码"按钮
2. 复制生成的代码
3. 打开 index.html，找到 playlist 部分
4. 用新代码替换旧的播放列表

五

提交到 GitHub

git add index.html music/
git commit -m "更新音乐"
git push origin main
